APPLiED

An AI-powered job tracking tool that matches your skills and resume against real listings using a LangGraph agent — so you apply smarter, not harder.

tech stack
Flask backend API server
LangGraph AI agent orchestration
PostgreSQL job & profile storage
Remotive API remote job listings
SerpApi Google Jobs scraping
Claude LLM for scoring & reasoning
API endpoints
GET /health server & db health check
GET /jobs?limit=N list jobs from db
POST /jobs/ingest fetch & save jobs from Remotive
GET /jobs/google/:user_id fetch Google Jobs for user preferences
POST /profiles create or update a user profile
POST /agent run the AI matching agent for a user
